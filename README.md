# Walmart Labs assessment
## How to install
```
npm install
```

## How to start the REST API
```
npm run serve
```

## How to consume it
```
URL: http://localhost:3001/api/stores-by-location/{zipCode}/{radius}
ex: http://localhost:3001/api/stores-by-location/33487/10
```

## How to test it
```
npm run test
```

## How to check its test coverage
```
npm run coverage
```
